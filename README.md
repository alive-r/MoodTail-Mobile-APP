**Setup Steps**  
1. Clone projects

    ```
    git clone https://github.com/alive-r/MoodTail-Mobile-APP.git
    cd MoodTail
    ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npm run ios
   npm run android
   ```
