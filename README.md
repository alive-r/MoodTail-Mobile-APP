**Setup Steps**  
1. Clone projects

    ```
    git clone https://gitlab.oit.duke.edu/ly202/ece59012_project.git
    cd MoodTail
    ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npm run ios
   npm run android
   ```
